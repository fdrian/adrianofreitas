Ferramentas para segurança e testes web

Base64 Debugger

Ferramenta para codificar e decodificar strings em Base64. Útil para trabalhar com dados binários em formato de texto.


JWT Debugger

Ferramenta para validar e decodificar JWT (JSON Web Tokens). Visualize o conteúdo do token e verifique sua validade.


URL Debugger

Esta ferramenta permite codificar ou decodificar texto usando percent-encoding (codificação de URL). Útil para depurar requisições HTTP, parâmetros e payloads com caracteres especiais.


Open Redirect Tester

Insira uma URL com um parâmetro de redirecionamento (ex: ?url=, ?next=) e testaremos contra payloads comuns de open redirect.
