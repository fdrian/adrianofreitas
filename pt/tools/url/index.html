Esta ferramenta permite codificar ou decodificar texto usando codificação percentual (URL encoding). Útil para depurar requisições HTTP, parâmetros de query e payloads com caracteres especiais.

Por que usar codificação de URL?

  • Escapar parâmetros que contenham espaços, &, =, ?, etc.
  • Injetar payloads com segurança em PoCs para testes de XSS ou SQLi.
  • Inspecionar redirecionamentos e URLs encadeadas durante testes de penetração web.
